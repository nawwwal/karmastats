🥇 Superiority Trial

Demonstrate that one treatment is better than another. Used to establish treatment efficacy with statistical superiority.

🚫 Non-inferiority Trial

Show that a new treatment is not unacceptably worse than standard care. Used for regulatory approval of alternative treatments.

⚖️ Equivalence Trial

Demonstrate that two treatments are therapeutically equivalent. Used for biosimilar and generic drug approval.

Superiority Trial Sample Size Calculator

▶ Statistical Formula & Methodology
Binary: n = 2(z(α/2) + z(β))²p̄q̄ / (p₁ - p₀)²
Continuous: n = 2(z(α/2) + z(β))²σ² / δ²

Where: n = sample size per group, z(α/2) = critical value for significance, z(β) = critical value for power, p₁ = treatment proportion, p₀ = control proportion, σ = standard deviation, δ = mean difference

Non-inferiority Trial Sample Size Calculator

▶ Statistical Formula & Methodology
n = 2(z(α) + z(β))²p̄q̄ / (d + Δ)²

Where: Δ = non-inferiority margin, d = expected true difference, α = one-sided significance level

Equivalence Trial Sample Size Calculator

▶ Statistical Formula & Methodology
n = 2(z(α/2) + z(β))²p̄q̄ / (Δ - |d|)²

Where: Δ = equivalence margin, |d| = absolute expected true difference, α = two-sided significance level
